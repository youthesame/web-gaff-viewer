# GAFF Viewer

GAFF力場（General Amber Force Field）の原子タイプとRESP部分電荷を可視化する、静的な3Dビューワーです。SYBYL mol2ファイルを読み込み、3Dmol.jsでインタラクティブに表示します。ビルド不要（npm/バンドラ不使用）、CDN依存のみで動作します。

## 使い方

1. 分子選択のドロップダウンから `BTBT` / `DTBDT` を選択、または右パネルの「ファイルを開く…」やビューワー領域へのドラッグ&ドロップで手持ちのmol2ファイルを読み込めます。
2. 表示モードを切り替えられます。
   - **GAFFタイプ**: 原子をGAFF原子タイプ（`ca`, `cc`, `ss`, `ha` など）ごとに色分けし、各原子の近くにタイプ名をラベル表示します。
   - **電荷**: RESP部分電荷の符号を色（負=赤系、正=青系、ゼロ付近=白）で、絶対値を球の大きさで表現します。ラベルには電荷値を表示します。
3. 「ラベル表示」トグルでラベルの表示/非表示を切り替えられます。
4. 右パネルの原子情報テーブルで全原子の情報（原子名・元素・GAFFタイプ・電荷）を確認できます。行をクリックすると、対応する原子が3Dビュー内でハイライトされます。

## ローカルでの起動方法

このディレクトリで簡易HTTPサーバを立てて開いてください（`fetch`によるmol2読み込みのため `file://` では動作しません）。

```bash
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000/` にアクセスします。

## ファイル構成

- `index.html` — ページ本体（3Dビューワー + サイドパネル）
- `style.css` — スタイル
- `js/app.js` — アプリロジック（3Dmol.js制御、色/ラベル/テーブル生成）
- `js/mol2.js` — mol2パーサ（GAFFタイプ・電荷の抽出）
- `mol2/*.mol2` — サンプル分子（BTBT, DTBDT）
